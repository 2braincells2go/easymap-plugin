Documentation du plugin "easyMap"

carte 1

$("#map-1").easymap();
Cette présentation minimale permet de créer une "Google Maps" de 400px sur 300, centré sur Paris.
Le zoom de base est de 8 (Le zoom pouvant aller de 1 à 21).
Les controles de la carte sont dans leur état par défaut.
La map ne contient aucun markers par défaut.

carte 2

$("#map-2").easymap({
	size: ['100%', '300px'],
	control: {
		zoom: 4,
		center: [46.60611, 1.87528]
	}
});
Dans cette présentation, les paramètres présentés sont :
  • La taille. Tableau à 2 éléments : [width, height]. Possibilité d'utiliser les 'px' comme les '%'
  • Le zoom (valeur pouvant aller de 1 à 21)
  • Le centre de la carte. Tableau à 2 éléments : [latitude, longitude].

    • carte 3

    $("#map-3").easymap({
	size: ['600px', '450px'],
	control: {
		zoom: 6,
		center: [46.60611, 1.87528],
		disableDefault: true,
		zoomControl: false,
		mapTypeControl: false
	}
});
    Ici, je rajoute 3 paramètres au niveau des controles de la map :
    • disableDefault: boolean. Les valeurs sont "true" ou "false" (false par defaut).
    • zoomControl: boolean. (true par defaut)
    • mapTypeControl: boolean. (true par defaut)

    Pour ajouter des markers sur la carte, il suffit d'ajouter la fonctionnalité "markers".
    Les valeurs d'un markeur sont les suivants :
    • Les coordonnées du marker (latitude et longitude). Valeur obligatoire.
    • Le nom de la ville (qui apparait dans un infowindow, au clic du marker). Valeur obligatoire.
    • L'icône du marker. Non obligatoire.
